# 🚀 AI Career Intelligence Platform

An **AI-powered career guidance platform** that analyzes resumes, identifies skill gaps, predicts employability scores, and provides personalized career recommendations using **Machine Learning and Generative AI**.

This project combines **React, Node.js, FastAPI, and AI models** to create a complete intelligent career development system.

---

# 🎯 Project Vision

The goal of this platform is to help students and job seekers:

* Analyze their **resume skills**
* Identify **missing skills for target jobs**
* Predict their **employability score**
* Receive **AI-powered career guidance**
* Explore **data-driven career paths**

The system integrates **Artificial Intelligence, Machine Learning, and Data Analytics** to provide personalized insights for career growth.

---

# 🧠 Core Features

## 📄 AI Resume Analyzer

Upload a resume (PDF) and automatically extract:

* Skills
* Education
* Experience
* Certifications

The system analyzes the resume and generates **AI-powered improvement suggestions**.

---

## 🧩 Skill Gap Analyzer

Compares user skills with industry job requirements.

Example:

```
User Skills:
Python, SQL

Target Role:
Data Scientist

Missing Skills:
Machine Learning
Statistics
Data Visualization
```

This helps users understand **what skills they need to learn next**.

---

## 🤖 AI Career Assistant

An intelligent chatbot powered by **Google Gemini AI** that answers career questions.

Example:

```
User:
Which skills should I learn to become a Data Scientist?

AI Response:
Recommended Skills:
• Python
• Machine Learning
• SQL
• Statistics
```

---

## 📊 Employability Score Prediction

Predicts a user's employability score using a **Machine Learning model**.

Input factors include:

* Number of skills
* Work experience
* Certifications

Example output:

```
Employability Score: 82 / 100
Strength: Technical Skills
Weakness: Communication
```

---

## 🧭 Career Simulation

Suggests possible career paths based on a user’s skills and job market demand.

Example:

```
Recommended Career Path:
Data Analyst → Machine Learning Engineer → Data Scientist
```

# 🧮 Algorithms Used

### Skill Matching

Uses **Set Intersection Algorithm** to compare user skills with job requirements.

Formula:

```
Match Score = |UserSkills ∩ JobSkills|
```

---

### Employability Prediction

Uses **Linear Regression** to predict employability score based on:

* skills
* experience
* certifications

---

### Career Recommendation

Uses a **rule-based recommendation system** based on job market data.
